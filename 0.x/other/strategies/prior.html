    Prior#

    A strategy where the probability of guessing a class is equal to the class's prior probability.

    Data Type: Categorical

    Parameters#

    This strategy does not have any parameters.

    Example#

    use Rubix\ML\Other\Strategies\Prior;

$strategy = new Prior();
